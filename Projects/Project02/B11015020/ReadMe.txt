### 安裝php(已獲得教授允許使用該語言)
```shell
sudo apt update -y && sudo apt install php -y
```
### 請確保您的環境已具備以下條件
- 該目錄下所有檔案、目錄已具有「可讀、可寫、可執行」的權限
- 已安裝「php」，且於終端機中輸入「php」是可以是別的指令
- 已具備root權限

### 執行方法
```shell
php ./main.php input1.pla output1.pla
php ./main.php input2.pla output2.pla
php ./main.php input3.pla output3.pla
```
